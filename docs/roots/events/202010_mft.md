# MakerFaireTokyo2020

2020年10月3日、4日に開催される[Maker Faire Tokyo 2020](https://makezine.jp/event/mft2020/)に出展しました。

今年の展示内容は2つです。

## コロナ対策を考慮した遠隔操縦体験

センサを使ってロボットを遠隔操縦し、PK対決をします。

![操作説明書](../../img/roots/events/mft2020_manual.png)

## Scrambleとのミニ試合

次世代ロボットエンジニア支援機構（通称：Scramble）のSSLチームと試合をします。
